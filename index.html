Project 1: Galton Board Game

Tool: Amp code

Description: An interactive simulation of the Galton Board, also known as a bean machine or quincunx. This project demonstrates the principles of probability and normal distribution through a visual and engaging web-based game. Users can drop balls from the top of the board and watch as they cascade through a triangular array of pegs, ultimately forming a bell curve distribution at the bottom.

Technologies used: HTML5, CSS3, JavaScript, SVG for graphics rendering

Project 2: 新北市里界地圖 (New Taipei City District Map)

Tool: ChatGPT + d3.js prompt

Description: An interactive map of New Taipei City's districts and villages. This project showcases the administrative divisions of New Taipei City using D3.js for data visualization. Users can explore the map, zoom in/out, and hover over different areas to see district and village names. The map features dynamic coloring and highlighting of districts and villages, providing an engaging way to visualize the city's geography.

Technologies used: HTML5, CSS3, JavaScript, D3.js for data visualization, Canvas for rendering, Tailwind CSS for styling
